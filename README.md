# 💰 Vanity Ledger

> 소비 내역과 감정을 함께 기록하는 개인 가계부 서비스

---

## 📌 프로젝트 개요

**Vanity Ledger**는 단순한 지출 관리를 넘어, 소비할 때의 감정과 위치 정보를 함께 기록하여 자신의 소비 패턴을 더 깊이 이해할 수 있도록 돕는 가계부 웹 애플리케이션입니다.

---

## ✨ 주요 기능

### 1. 🔐 로그인 / 회원가입

- 아이디 / 비밀번호 기반 로그인
- 회원가입 시 닉네임, 아이디, 비밀번호, 거주 지역 입력
- 로그인 상태는 localStorage로 세션 유지

### 2. 📝 수입 / 지출 기록

- 지출 / 수입 모드 전환
- 날짜, 금액, 메모 입력
- 지출 시 감정 선택 (😊 만족 / 😔 후회)
- 지출 시 소비 위치 선택 (서울 구 단위)
- 지출 시 카테고리 선택 (수입은 자동 분류)

| 카테고리  | 카테고리 | 카테고리 |
| --------- | -------- | -------- |
| 🛍️ 쇼핑   | 🛵 배달  | 🍽️ 식당  |
| 🏪 편의점 | ☕ 카페  | 🍺 술집  |
| 🧴 생필품 | 🚌 교통  | 🏥 병원  |
| 📦 기타   |          |          |

### 3. 📊 통계

- 주간 / 월간 전환 및 기간 이동
- 수입 / 지출 / 순수익 합계
- 감정 통계 (만족 / 후회 비율, 프로그레스 바)
- 거래 내역 목록 (최신순)
- 지역별 소비 집계 (금액 순위)
- 카테고리별 지출 집계
- PDF 내보내기 (브라우저 인쇄)

### 4. 🗺️ 지도

- 서울 구 단위 소비 위치를 네이버 지도에 핀으로 표시
- 구 선택 시 해당 지역 거래 상세 내역 확인
- 우측 패널에서 지역별 소비 요약 및 통계 제공

### 5. 🏅 칭호 & 뱃지 시스템

월말에 소비 패턴을 분석하여 뱃지가 자동 부여됩니다. 획득한 뱃지 중 하나를 칭호로 선택할 수 있습니다.

| 뱃지 (영문 key)    | 조건                              |
| ------------------ | --------------------------------- |
| 🏠 Home Potato     | 거주지 소비 비율 80% 이상         |
| 💸 Money Splash    | 월 총 지출 100만 원 이상          |
| 🪙 Coin Guardian   | 월 총 지출 20만 원 이하           |
| 🍻 Tipsy Champ     | 술집(bar) 지출 비율 50% 이상      |
| 🤒 Fragile Hero    | 병원(hospital) 지출 10만 원 이상  |
| ☕ Coffee Zombie   | 카페(cafe) 지출 20만 원 이상      |
| 🗺️ Lost Explorer   | 거주지 소비 비율 20% 이하         |
| 🛵 Delivery Addict | 배달(delivery) 지출 비율 30% 이상 |

### 6. 👤 프로필

- 닉네임 / 거주 지역 수정
- 활동 요약 (총 거래 수, 총 지출, 총 수입)
- 획득한 뱃지 컬렉션 확인
- 칭호 선택 및 변경
- 로그아웃

### 7. 🎨 감정 기반 UI 테마

이번 달 감정 통계에 따라 UI 스타일이 동적으로 변경됩니다.

| 상태    | 설명                      |
| ------- | ------------------------- |
| 😔 후회 | 스타일 최소화 (기본 HTML) |
| 😐 평범 | 기본 상태                 |
| 😊 만족 | 강화된 CSS 적용           |

## 🖥️ 페이지 구성

| 경로        | 페이지      | 설명                                        |
| ----------- | ----------- | ------------------------------------------- |
| `/login`    | 로그인      | 아이디/비밀번호 로그인                      |
| `/signup`   | 회원가입    | 신규 회원 등록                              |
| `/`         | 홈 대시보드 | 만족 지수, 수입/지출/순수익 요약, 최근 거래 |
| `/register` | 거래 등록   | 지출/수입 기록 입력                         |
| `/stats`    | 통계        | 주간/월간 소비 분석, PDF 내보내기           |
| `/map`      | 지도        | 구 단위 소비 위치 시각화                    |
| `/profile`  | 프로필      | 뱃지/칭호 관리, 정보 수정                   |

## 🛠️ 기술 스택

| 구분        | 기술                                      |
| ----------- | ----------------------------------------- |

## 🚀 시작하기

### 사전 요구사항

- Node.js 설치
- npm 설치

### 설치 및 실행

```bash
# 저장소 클론
git clone https://github.com/your-username/ledger.git
cd ledger

# 의존성 설치
npm install

# json-server 실행 (별도 터미널, 포트 3000)
npx json-server --watch db.json --port 3000

# 개발 서버 실행
npm run dev
```

## 📁 프로젝트 구조
