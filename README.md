# 💰 Ledger

> 소비 내역과 감정을 함께 기록하는 개인 가계부 서비스

---

## 📌 프로젝트 개요

**Ledger**는 단순한 지출 관리를 넘어, 소비할 때의 감정과 위치 정보를 함께 기록하여 자신의 소비 패턴을 더 깊이 이해할 수 있도록 돕는 가계부 웹 애플리케이션입니다.

---

## ✨ 주요 기능

### 1. 🔐 로그인 / 회원가입

- 아이디 / 비밀번호 기반 로그인
- 회원가입 시 닉네임, 아이디, 비밀번호, 거주 지역 입력

### 2. 📝 수입 / 지출 기록

- 날짜, 금액, 메모 등 세부 정보 입력
- 기록 당시의 감정 선택 (😊 행복 / 😔 우울)
- 위치 정보 기록 (네이버 지도 API, 구 단위)
- 카테고리 선택

| 카테고리  | 카테고리 | 카테고리 |
| --------- | -------- | -------- |
| 🛍️ 쇼핑   | 🛵 배달  | 🍽️ 식당  |
| 🏪 편의점 | ☕ 카페  | 🍺 술집  |
| 🧴 생필품 | 🚌 교통  | 🏥 병원  |
| 📦 기타   |          |          |

### 3. 🔍 거래 내역 조회

- 날짜 / 주간 / 월별 / 기간별 필터링
- 카테고리별 필터링
- 감정별 필터링
- 조건에 맞는 요약 내역 조회

### 4. 📊 월별 재정 요약 및 분석

- 월별 수입, 지출, 순이익 요약
- 감정 통계 (행복 / 우울 비율)
- 지역별 소비 합계 (동/구 단위)
- 보고서 내보내기 (PDF / CSV)

### 5. 🏅 칭호 & 뱃지 시스템

소비 패턴에 따라 칭호와 뱃지가 부여됩니다.

| 칭호           | 조건                                  |
| -------------- | ------------------------------------- |
| 🏠 집순이      | 소비 구역의 80% 이상이 거주지         |
| 💸 플렉스      | 지난달 소비 100만 원 이상             |
| 🪙 짠돌이      | 지난달 소비 20만 원 이하              |
| 🍻 초빼이      | 술집 카테고리가 한 달 소비의 50% 이상 |
| 🤒 유리몸      | 최근 한 달 병원비 10만 원 이상        |
| ☕ 카페인 중독 | 최근 한 달 카페 소비 20만 원 이상     |
| 🗺️ 방랑자      | 거주지 소비가 전체의 20% 이하         |

### 6. 🗺️ 지도

- 결제가 이루어진 위치를 지도에 핀으로 표시
- 핀 클릭 시 해당 거래 상세 내역 확인

### 7. 🎨 감정 기반 UI 테마

감정 통계에 따라 UI 스타일이 동적으로 변경됩니다.

| 상태      | 설명                      |
| --------- | ------------------------- |
| 😔 우울함 | 스타일 최소화 (기본 HTML) |
| 😐 평범   | 기본 상태                 |
| 😊 행복   | 강화된 CSS 적용           |

## 🚀 시작하기

### 사전 요구사항

- Node.js 설치
- npm 또는 yarn 설치

### 설치 및 실행

```bash
# 저장소 클론
git clone https://github.com/your-username/ledger.git
cd ledger

# 의존성 설치
npm install

# json-server 실행 (별도 터미널)
npx json-server --watch db.json --port 3000

# 개발 서버 실행
npm run dev
```
